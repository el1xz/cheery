# cheery

## Project setup
```
git clone
npm install

If you encounter an error "blocked by CORS policy"
need install Moesif Origin & CORS Changer and turn on it

```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
